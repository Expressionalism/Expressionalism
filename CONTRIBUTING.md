## How to Contribute
1. Fork the repo.
2. Create a branch (e.g., `feature/new-toggle`).
3. Make changes: Add toggles, examples, fixes, or refinements.
4. Test locally (e.g., run the script).
5. Submit a pull request with a description.

Respect licenses (MIT for code, CC BY 4.0 for docs). Questions? Open an issue!
